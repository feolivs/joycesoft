# Dashboard Administrativa JoyceSoft

Dashboard administrativa moderna e responsiva desenvolvida com React, TypeScript, Tailwind CSS, Zustand e Framer Motion.

## 🚀 Tecnologias

- React
- TypeScript
- Tailwind CSS
- Zustand (Gerenciamento de Estado)
- Framer Motion (Animações)
- React Router DOM
- Headless UI
- Heroicons

## 📋 Funcionalidades

- Dashboard com cards de estatísticas
- Gerenciamento de clientes (CRUD)
- Interface responsiva
- Animações suaves
- Tema personalizado
- Integração com API (mockada)

## 🔧 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/dashboard.git
cd dashboard
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

## 🎨 Estrutura do Projeto

```
/src
├── components/     # Componentes reutilizáveis
├── pages/         # Páginas da aplicação
├── store/         # Gerenciamento de estado (Zustand)
├── services/      # Serviços e integrações
└── App.tsx        # Componente principal
```

## 🔄 Integração com API

O projeto está configurado para se comunicar com uma API. Atualmente, está usando dados mockados, mas pode ser facilmente integrado com uma API real.

Para integrar com sua API:

1. Atualize o arquivo `src/services/api.ts`
2. Configure a URL base da API
3. Implemente os métodos de comunicação
